Version Control via Git

Why

  • 易於追蹤檔案變更
  • 易於團隊協力開發

Git

  1. 安裝 git

  2. 檢查 git 是否安裝成功

  3. 初始化 Git repository

  4. 確認 working tree 狀態

  5. 將所有檔案加入 index

  6. 記錄 repository 的變更

  7. 顯示變更記錄

Github

  1. 登入 Github 及 創建新的 Repository

  2. 將 repository 加入 remote

  3. 將 local 的記錄更新至 remote repository
